# Простая сборка gulp
c плагинами для оптимизации js, и предпроцессором Sass.

## Структура каталогов
>./src/styles/\*\*/*.sass
>./src/scripts/\*\*/*.js

## Инстукция

1. Скачать в любую папку
2. В терминале ввести npm i 
3. Для запуска дефолтной щадачи с отслеживанием ввесте в терминале gulp
4. Enjoi
